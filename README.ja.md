# MikuMikuWorld for UntitledCharts
[MikuMikuWorld](https://github.com/crash5band/MikuMikuWorld)の[UntitledCharts](https://untitledcharts.com/)用フォーク。

## 追加された機能：
MikuMikuWorldの元の機能に加え、以下の機能が追加されています：
- ダメージノーツ
- 拡張レーン
- ガイドの色付け
- 新しいイ－ジング
- ハイスピードの補間
- スライドの中継点の繰り返し
- ノーツプロパティパネル
- 小数点のノーツレーンと幅のサポート
- などなど
